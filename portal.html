⚙ Setup checklist

✓ Step 1 — Add your Formspree form ID

Orders from customers submit to Formspree (free for up to 50 submissions/month). Sign up, create a form, and copy your Form ID (looks like xyzabcde). Then replace REPLACE_WITH_FORM_ID in two places:

A
In order.html, find the line:
action="https://formspree.io/f/REPLACE_WITH_FORM_ID"
and replace REPLACE_WITH_FORM_ID with your actual ID.
B
In your Formspree dashboard, set the notification email to the shop's email address so you get an alert for every new order.

✓ Step 2 — Add real store hours to order.js

Open assets/js/order.js and update these two lines at the top to match your actual opening and closing times:

const STORE_OPEN_HOUR = 6;  /* 6:00 AM */
const STORE_CLOSE_HOUR = 14; /* 2:00 PM */

Hours use 24-hour format. For example, 8:00 AM = 8, 5:00 PM = 17.

✓ Step 3 — Add prices to menu-data.js

Open assets/data/menu-data.js and fill in the price: field for each item. For example: { name: "Chocolate Iced Donut", price: "1.25" }. Prices will then show in the cart and on the order confirmation.

✓ Step 4 — Change the portal passcode

The default passcode is donuts. To change it, open portal.html, find the line near the bottom that reads:
if (input === 'donuts')
and replace donuts with your chosen passcode. Note: this is a front-end deterrent only — for real security, ask your developer to add server-side authentication.

🔗 Step 5 — Add "Order Ahead" to your nav (optional)

To add the Order Ahead page to the main navigation, open every .html file in your site root and add this line inside the <nav class="main-nav">:

<a href="order.html">Order Ahead</a>

You can also replace the existing "Order Ahead" button on index.html to link directly to order.html instead of the #order-ahead anchor.

🚀 Future upgrade: Stripe payments

When you're ready to accept payment online (instead of at pickup), a developer can add Stripe Checkout to the order page. This requires a small serverless backend function (Netlify Functions work well with your current hosting approach — free tier available). The order.js file is already structured to support this upgrade cleanly.
